Event design and event planning work hand in hand to deliver a successful event.Events by Knight offer tailor made events. This means we incorporate both event design and event planning to create that perfect event.

The time to plan events varies from a few weeks to months. If your event is small, we start planning 3 days before. If your event is too large, we start planning immediately after booking. Because, there are a lot of things to do.
